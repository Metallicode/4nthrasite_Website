Rhythm Machines

KIK

Industrial Kick Engine

Drum Synth

A precision kick drum synthesizer built around layered synthesis stages. The core generates a pitched transient from a tunable oscillator with pitch envelope; the transient & noise layer adds attack click and body noise; the destruction & rumble stage applies saturation and low-end resonance for industrial weight.

A 16-step pattern sequencer with accent and reverse modes controls when the engine fires.

Key controls

Pitch start/end  ·  Decay  ·  Waveform  ·  Click level  ·  Noise colour  ·  Drive  ·  Sub resonance  ·  Accent steps  ·  Fill probability

EUCLIDEAN

Euclidean Rhythm Generator

Drum Synth · Sampler · Sequencer

A 6-track polymetric drum machine that distributes hits evenly across steps using the Euclidean (Bjorklund) algorithm. Each track shows its pattern as a necklace of dots around a circle — active hits glow in the track colour, the current step is highlighted with a clockhand needle.

Each track has its own step count (K), so tracks naturally loop at different lengths, creating evolving polymetric grooves. Each hit triggers a synthesised drum voice with carrier oscillator, FM modulator, AM modulator, noise layer, wavefold distortion, and resonator. Sample override per track.

Key controls

Hits (N)  ·  Steps (K)  ·  Offset (R) per track  ·  12 synth parameters per voice  ·  Sample mode

NL SEQ

Non-Linear Drum Sequencer

Sequencer

A 6-channel drum sequencer where a drawn tempo curve controls how fast or slow each individual step plays. The curve runs FAST (top) to SLOW (bottom) and is drawn directly on a canvas above the step grid.

The BAL button calculates timing deficit from drawn changes and randomly compensates by speeding up untouched steps — keeping the total loop length equal to the global BPM.

Key controls

Curve draw tools (pen / straight line)  ·  Curve presets (flat, ramp, sine, accel, decel, random)  ·  BAL button  ·  8–64 steps  ·  Per-channel synth

CELLS

Cellular Automata Sequencer

Sequencer · Generative

An 8-track drum sequencer driven by Langton's lambda cellular automaton rule. Each of the 16 cells holds a state 0–7 mapping to a rhythmic subdivision: 0=silent, 1=single hit, up through triplets, quadruplets, and 32nd-note bursts.

The CHAOS knob sets lambda — how ordered or random the CA's evolution is. Patterns mutate every N loops; any track can be frozen. Click cells to manually set states.

Key controls

CHAOS (λ)  ·  RATCHET probability  ·  EVOLVE RATE  ·  DENSITY  ·  Per-track freeze  ·  Per-track sample loading

BEATS

Polymetric Beats

Sequencer · Polyrhythm

A 6-channel step sequencer where each channel runs at its own independent step count (4–32), naturally producing polyrhythmic phase drift. A 7-step channel against a 5-step channel creates a 35-step composite before repeating.

Per-step pitch nudge, just-intonation snap, and swing control allow fine timing and harmonic shaping of each channel.

Key controls

Per-channel step count  ·  Pitch offsets per step  ·  JI snap  ·  Swing  ·  Per-channel sample loading

SNAKE

Snake Sequencer

Sequencer · Game

An arcade-style sequencer where you steer a snake across a grid with arrow keys. The grid is seeded with coloured cells mapped to 4 synth slots. As the snake consumes cells it records a sequence of sound events; Play loops back through the recorded trail.

Bit-crush and sample-rate reduction quantizer for lo-fi textures. Pattern generator seeds the grid with interesting initial configurations.

4 ADSR synth slots  ·  Arcade presets  ·  Loop mode  ·  Bit-crush  ·  Pattern generator

Synths

T-303

Turing 303

Synth · Sequencer

A TB-303 style acid synthesizer driven by a 16-bit Turing machine shift register. At 0% probability the register loops identically (stable pattern); at 50% it randomly flips bits (maximum chaos); at 100% it mirror-flips.

Click the LED bit-row to write pitch values directly. Features sawtooth or square waveforms, resonant 303-style low-pass filter with envelope modulation, portamento, accent, and reverb.

Key controls

Probability  ·  Seq length 1–16  ·  Filter resonance / envelope / decay  ·  Portamento  ·  Accent level  ·  LED pitch editor

PENDULUM

Pendulum Drone

Synth · Physics

A drone synthesizer whose timbre is controlled by a simulated multi-arm pendulum. Each arm's angular position modulates an audio parameter (filter cutoff or wavefolder drive). Drag the bob to set it in motion; hold to dampen.

Arms can be equipped with motors to sustain oscillation indefinitely, creating self-modulating evolving drones.

Key controls

Arm count  ·  Modulation target per arm  ·  Motor on/off  ·  Drag (mouse)

EDO ARP

Equal Division Arp Synth

Synth · Sequencer

A synthesizer tunable to any equal division of the octave (EDO) — from 5-EDO to 31-EDO and beyond. The keyboard dynamically redraws for the selected EDO, accurately rendering microtonal pitches.

Notes can be played as arpeggio (up / down / up-down / chord / random) or programmed in a 303-style step sequencer.

Key controls

EDO selector  ·  Arp mode  ·  Step sequencer (Seq 303 mode)

TENSION

Fusion Architect

Synth · Generative

A 4-operator FM synthesizer driven by a stochastic chord navigation engine. A puck moves through a 2D tension space selecting chord targets that feed pitch into the FM operators. Selectable FM algorithms; operator 4 has self-feedback.

SCATTER mode maximises harmonic unpredictability, pushing the puck to distant chord targets and creating dissonant, evolving FM textures.

Key controls

TOPO

Topology Sequencer

Sequencer · Visual

A spatial sequencer on a 35×37 deformable mesh grid. Rows = MIDI pitch; columns = time slices. A smudge brush distorts the mesh in real time; the sequencer reads pitch from a selected column — deformation above/below rest maps to semitone pitch shifts.

Moving between columns changes the melodic contour entirely, allowing you to sculpt a landscape of different melodies and navigate between them.

Key controls

Smudge brush  ·  Column selector  ·  Base note  ·  16 steps

Draw melodies as strokes or create rich spectral textures through layered colour. Different brush colours map to different timbres.

SPACE

Neural Topology Synth

Synth · Analysis

Loads an audio sample and extracts 8 spectral "engrams" — FFT frequency snapshots from evenly-spaced moments. These are distributed across a visual neural network of hundreds of nodes.

Moving the mouse probe blends nearby engrams in real time, driving 40 simultaneous oscillator voices. The entire timbral space is defined by your source audio.

Key controls

Harmony & Melody

GEO HAR

Geometric Harmony Sequencer

Sequencer · Harmony

A timeline canvas where chord blocks are placed by clicking. A block's horizontal position and width sets start time and duration; vertical position and height sets pitch range. A playhead sweeps left to right triggering polyphonic chords.

Double-click to add a block, drag to move, resize from edges. Just-intonation snap aligns pitches to pure harmonic ratios.

TONNETZ

Tonnetz Grid

Harmony · Recorder

An interactive Neo-Riemannian Tonnetz — a hexagonal lattice where pitch classes are arranged so harmonically close notes sit physically close. Right = perfect fifth; diagonal = major/minor third. Any triangle of adjacent nodes forms a triad.

Hover to preview pitches; click nodes to record a path that plays back as melody or voice-leading.

Key controls

Hover preview  ·  Click-record  ·  Playback direction  ·  Tempo

Hover to hear; click to record into a looping sequence. Playback direction: forward, backward, or ping-pong.

Up to 12 scale degree slots; scan modes: random, left-to-right, top-to-bottom. Sample override per slot.

Granular

GRANULAR

Granular Synth

Granular · Sampler

A classic granular synthesis engine. Load any audio file; a read-position pointer moves through the sample. The engine continuously fires small grains; SIZE, DENSITY, and envelope shape determine texture.

DRIFT causes the read position to wander. Grains can play forward, reverse, or alternating. Pitch transposition is independent of playback speed.

Key controls

Sample loading  ·  Read position  ·  Grain size  ·  Density  ·  Drift  ·  Direction  ·  Pitch transpose

GRN MTX

Matrix Granular

Granular · Sampler

A granular synthesizer built around a scrollable grid matrix. Horizontal = time, vertical = pitch offset. Paint cells to arm grain-trigger positions; a moving playhead fires grains at armed cells from a loaded audio sample.

SCATTER randomises read position for cloud-like texture. Transient detection intelligently maps sample regions to the pitch axis.

Key controls

Grid painting  ·  Grain size  ·  Scatter  ·  Sample loading  ·  Playhead

LOOPER

Live Loop Recorder

Sampler · FX

A 5-channel live looper with integrated step sequencer. Each channel records from any audio input and plays back as a continuous loop. Channel 1 is master; SYNC auto-stops recording on channels 2–5 at the master loop length.

Each channel has a waveform display with draggable start/end trim markers (down to 5ms). A 16-step sequencer at the bottom triggers any channel per step, synced to global BPM.

FX & Ambience

SHOEGAZE

Shoegaze Processor

FX · Ambience

A live audio processor for lush, cascading ambience. Processes mic or file input through three stages: granular pitch shifter (harmonic colour), feedback delay network (dense diffuse reverb), and shimmer (pitch-shifts reverb up an octave and feeds it back into the FDN).

HOLD freezes the current wet tail indefinitely, sustaining infinite reverb pads.

Key controls

Granular pitch shift  ·  FDN reverb size / decay  ·  Shimmer amount  ·  HOLD button  ·  Wet/dry

SPECTRAL

Spectral Editor

Additive · Analysis

A partial-based additive synthesizer with a visual editor. Load audio to extract sinusoidal components via FFT partial tracking — each partial appears as a dot on the frequency/amplitude canvas.

Drag partials to change frequency or amplitude; draw new ones from scratch; delete existing ones. Playback resynthesizes the current partial set in real time.

Key controls

FFT analysis  ·  Partial drag  ·  Multi-select  ·  Add / delete partials  ·  Real-time resynthesis
